.Shape处于激活状态
  .谁能让他处于激活状态
   .ToolBox拖动上去
   .点击Shape
   .概要面板点击
  .通信方式
   .Events.trigger(,{pageId,componentId})
   .App注册on
   .trigger到App上
  .激活后的操作
   .上一个激活的元素不激活
   .元素边框的变化
   .属性面板初始化
   .概要面板初始化
   .激活元素的z-index为最大层级数
   .可以resize

.Component放入Page的原则
 .从左侧拖入
  .ReactDOM.render(
     <Component.Component
       pageId={pageId}
       componentId={componentId}
       number={ShapeModel.getShapesByPage(pageId).length + 1}
       property={ShapePropertyDefaultConfig}
       getInstance={(ins) => {
         ShapeModel.add(ins);
       }}
     />, el
   ) 放入
 .有数据载入
   .ReactDOM.render(
      <Component.Component
        pageId={pageId}
        componentId={componentId}
        number={ShapeModel.getShapesByPage(pageId).length + 1}
        property={ShapePropertyDefaultConfig}
        getInstance={(ins) => {
          ShapeModel.add(ins);
        }}
      />, el
    ) 放入
  所以Component的初始化Props是不会再被外部改变的
  这时还需要和Property进行双向的数据同步，通过trigger或者调用函数的方式进行数据同步，建议用state，所有数据都放入state中
